      Cкрипт для создания галереи изображений по массиву данных. В HTML есть список ul.gallery.
    • Используется массив объектов images для создания элементов img вложенных в li.
    • Для создания разметки используются шаблонные строки и метод insertAdjacentHTML().
    • Все элементы галереи добавляются в DOM за одну операцию вставки.

      • Минимальное оформление галереи флексбоксом через CSS классы.
