    Cкрипт управления формой логина
  1. Обработка отправки формы form.login-form по событию submit.
  2. При отправке формы страница не перезагружается.
  3. Если в форме есть незаполненные поля, выводится alert с предупреждением о том, что все поля должны быть заполнены.
  4. Если пользователь заполнил все поля и отправил форму, значения полей собираются в обьект, где имя поля будет именем свойства,
    а значение поля - значением свойства. Для доступа к элементам формы используется свойство elements.
  5. Выведится объект с введенными данными в консоль и очищаются значения полей формы методом reset.
